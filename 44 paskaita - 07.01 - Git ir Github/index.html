Git ir Github

Git'as - populiariausia kodo versijavimo programa.
GitHub'as - vienas populiaresnių versijuojamo kodo talpinimo platformų.

Git komandos

  • init - sukuriama git repozitorija dabartiniame aplankale
  • status - nurodo informaciją apie lokalios git'o repozitorijos statusą
  • add - prideda nurodytus failus prie commit'inimui paruoštų failų (stage'ina)
  • rm --cached || restore --staged - išima nurodytus failus iš stage'inimo zonos (unstage'ina)
  • commit -m 'zinute' - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutė turėtų būti trumpa (ne ilgesnė nei 50 simbolių) ir aiški)
  • restore - sugrąžina failą į buvusio commit'o versiją (panaikina visus pakitimus darytus po paskutinio commit'o)
  • push - nusiunčia lokalios repozitorijos commit'us į išorinę repozitoriją
  • pull - atsiunčia išorinės repozitorijos commit'us į lokalią repozitoriją
  • clone
  • log
  • checkout
  • branch
